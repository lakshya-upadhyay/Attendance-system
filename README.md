# 📋 Attendance System

A simple, browser-based **Student Attendance Management System** built with pure HTML, CSS, and JavaScript — no server or database required.

---

## 🚀 Live Demo

Open `index.html` directly in any modern browser — no installation needed.

---

## 📌 Features

- Mark student attendance (Present / Absent)
- View and manage student records
- Clean and responsive UI
- Works entirely in the browser (no backend required)
- Lightweight — single HTML file

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| HTML5 | Structure & layout |
| CSS3 | Styling & responsiveness |
| JavaScript (Vanilla) | Logic & interactivity |

## ⚙️ Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Edge, Safari)

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/lakshya-upadhyay/Attendance-system.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd Attendance-system
   ```

3. **Open in browser**
   ```bash
   open index.html
   # or just double-click index.html in your file explorer
   ```

That's it — no `npm install`, no server setup, no database config. 🎉
